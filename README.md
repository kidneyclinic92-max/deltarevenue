# Delta Revenue Partners – Medical Billing Website

A professional, responsive website for Delta Revenue Partners: medical billing and revenue cycle management (RCM) services, inspired by [Delta Revenue Partners](https://deltarevenuepartners.com/).

## Hero background video

Place your hero video in **`public/assets/`**:

- **`hero-video.mp4`** (required) – main format
- **`hero-video.webm`** (optional) – for better support in some browsers

The hero uses the video as a full-bleed background with a dark overlay so text stays readable. If the files are missing, the section still shows the dark overlay and content.

## Features

- **Hero** with background video, specialty selector, and key metrics (cash flow, collection rate, days in A/R, success rate)
- **Services** – Medical Billing, Coding, DME, Hospital Billing, Practice Audit, Credentialing, Telehealth, Patient Help Desk
- **Why RCM** – GO FAST, GO BIG, GO BOLD, GO SMOOTH
- **Solutions by specialty** and **onboarding** sections
- **Benefits** and **Cost Calculator** CTAs
- **Centric One** product section and **testimonial**
- **Contact form** – Get Solution in 24 Hours
- Responsive layout and mobile menu

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Fonts: Outfit (headings), DM Sans (body)

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

Build output is in `dist/`.
